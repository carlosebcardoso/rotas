## Requisitos

- Python 3.10 ou superior

## Configuração do ambiente

### Linux / macOS

```bash
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

### Windows

```bash
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
```

## Desativando o ambiente virtual

```bash
deactivate
```
